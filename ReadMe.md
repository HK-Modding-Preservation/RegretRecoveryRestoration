# RegretRecoveryRestoration

This Randomizer connection restores the ability to retrieve your shade when egg shop replaces Jiji's normal function.

One item, the Regret Recovery, is added to your rando. When obtained, it will spawn your shade if one exists and remove it from wherever you previously died. 
It can be set to appear either at a random location or at egg shop. If placed at egg shop, the group setting will be ignored and the cost will be 1 egg, regardless of Rando's egg cost settings. 
If egg shop is not enabled, this item will not be placed.
